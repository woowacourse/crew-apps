# 우아한테크코스 프로덕트 빌더들

우아한테크코스 크루들이 만든 앱을 한눈에 모아 보는 웹 디렉토리입니다.

## 주요 기능

- **앱 카탈로그** — 크루들이 만든 12개 앱을 카드 형태로 탐색
- **검색** — 앱 이름, 설명, 제작자로 검색
- **카테고리 필터** — 생산성, 학습, 게임, 유틸리티, 커뮤니티, 라이프스타일
- **앱 상세 페이지** — 설명, 사용법, 스크린샷, 댓글

## 기술 스택

- React 18 + TypeScript
- Vite 5 (SWC)
- Tailwind CSS 3 + shadcn/ui
- React Router v6

## 시작하기

```bash
bun install
bun run dev        # http://localhost:8080
```

## 스크립트

| 명령어 | 설명 |
|--------|------|
| `bun run dev` | 개발 서버 실행 |
| `bun run build` | 프로덕션 빌드 |
| `bun run lint` | ESLint 실행 |
| `bun run test` | 테스트 실행 |
